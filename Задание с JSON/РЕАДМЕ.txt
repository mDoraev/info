ЗАДАНИЕ:
В админке на странице товара имеется вкладка "Адрес" (см скрин)
При открытии страницы необходимо заполнить селекты county, town, town_area и street из JSON 
То есть сначала удаляем все что там есть, потом заполняем. значение опции - поле option_id из джейсона, текст опции - поле name
Также вверху добавляем пустую опцию "Выбрать".
Среди полей есть selected - какая опция выбранная, ее и делаем выбранной. Если не выбрана ни одна, то ставим выбранной "Выбрать".
Для селектов town и street значение обязательно (то есть опция "Выбрать" должна быть disabled)

При выборе значения опции в селекте вызывывать урл, находящийся в переменной ajaxUrl и передавать на него 2 параметра (1 - code= (counrt, city, town_area или street); 2 id = -

ВАЖНО: поля id и option_id в JSON - не одно и то-же, часто одинаковы, но не всегда. И в select в качестве value заполняем option_id, а на урл передаем id (то есть, например, создает у опции какой-нибудь атрибут и в нем сохоаняем поле)
