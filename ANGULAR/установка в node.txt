узнать версию нода
должна быть последняя
node -v 

создать папку, войти в нее
создать можно и на компе, и командой mkdir papka

войти в папку
cd papka

установить глобально (-g) на комп ангуляр. нужно только в первый раз,
затем не нужно

npm install -g @angular/cli

узнать версию ангуляра
ng -v

создать приложение ангуляр (находясь в нужной папке)
ng new new-pril

зайти в папку приложения
cd new-pril

запустить сервер
ng serve

запустить сервер и открыть
ng serve -о
