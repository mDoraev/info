Галерея виджетов




			
обзор



Виджет галереи jQuery реализует область контента с изображениями, организованными в блоки предварительного просмотра и миниатюры.



На следующем рисунке показано отображение изображения на странице продукта с помощью виджета галереи:






Кроме того, виджет лупы можно использовать для демонстрации изображений в масштабе 100% в отдельном выделенном слое, а полноэкранный режим галереи можно использовать для навигации по всей полноразмерной фотографии.



Галерея отображается одинаково во всех поддерживаемых браузерах и является отзывчивой - она ​​правильно масштабируется на мобильных устройствах и настольных ПК.



Виджет галереи использует виджет Fotorama .



Важной особенностью реализации виджета галереи является возможность настройки параметров виджета в view.xmlфайле конфигурации темы.



Исходный код виджета галереи - lib / web / mage / gallery / gallery.js .



Источником лупы для галереи является lib / web / magnifier / magnify.js .



содержание



Инициализировать виджет галереи



Виджет галереи инициализируется, как описано в разделе «Инициализация JavaScript» .



Пример декларативной инициализации:



<Magento_Catalog_module_dir>/view/frontend/templates/product/view/gallery.phtml

копия



<script type="text/x-magento-init">
    {
         {
        "<element_selector>": {
            "mage/gallery/gallery": {
                "data": [{
                    "thumb": "<small_image_url>",
                    "img": "<small_image_url>",
                    "full": "<small_image_url>",
                    "caption": "<message>",
                    "position": "<number>",
                    "isMain": "<true/false>"
                }],
                "options": {
                    "nav": "<false/thumbs/dots>",
                    "loop": <true/false>,
                    "keyboard": <true/false>,
                    "arrows": <true/false>,
                    "allowfullscreen": <true/false>,
                    "showCaption": <true/false>,
                    "width": <number>,
                    "thumbwidth": <number>,
                    "thumbheight": <number>,
                    "height": <number>,
                    "transitionduration": <number>,
                    "transition": "<slide/crossfade/dissolve>",
                    "navarrows": <true/false>,
                    "navtype": "<slides/thumbs>",
                    "navdir": "<horizontal/vertical>"
                },
                "fullscreen": {
                    "nav": "<false/thumbs/dots>",
                    "loop": <true/false>,
                    "navdir": "<horizontal/vertical>",
                    "arrows": <true/false>,
                    "showCaption": <true/false>,
                    "transitionduration": <number>,
                    "transition": "<slide/crossfade/dissolve>"
                },
                "breakpoints": {
                    "<breakpoint_name>": {
                        "conditions": {
                            "max-width": "767px"
                        }
                        "options": {}
                    }
                }
            }
        }
    }
</script>

Опции



Для виджета галереи доступны следующие группы параметров:



    
  
  • data
    • 
  
  • options
    • 
  
  • fullscreen
    •

Все варианты для каждой группы описаны далее.



data



Массив изображений для отображения.



Где imageнаходится объект с ключами:



{
    "thumb": "<small_image_url>",
    "img": "<small_image_url>",
    "full": "<small_image_url>",
    "caption": "<message>",
    "isMain": "<true/false>"
}

копия



thumb, imgИ fullполные пути к собственно графическим файлам.



data/caption



Установите подпись для определенного изображения в dataполе. Пример инициализации во время выполнения с captionуказанным параметром:



galleryInstance({
    data: [
        {
            img: 'preview_number_one.jpg',
            thumb: 'preview_number_one.jpg',
            caption: 'T-Shirt Women side'
        }
    ]
});

копия



options


Набор параметров, доступных для области предварительного просмотра.



options/allowfullscreen


Показать кнопку, которая переключает полноэкранный вид галереи.



Тип : логическое значение



options/arrows



Отобразить навигационные стрелки на миниатюрах сторон.



Тип : логическое значение



options/height



Высота блока предварительного просмотра в пикселях или процентах.



Тип : Число, Строка



Значение по умолчанию : ноль



options/keyboard



Включить / выключить навигацию с помощью клавиатуры.



Тип : логическое значение



Значение по умолчанию :false



options/loop



Определите, отображаются ли изображения в цикле.



Тип : логическое значение



Значение по умолчанию :false



options/maxheight



Максимальная высота блока предварительного просмотра в пикселях или процентах.



Тип : Число, Строка



Значение по умолчанию :null



options/maxwidth



Максимальная ширина блока предварительного просмотра в пикселях или процентах.



Тип : Число, Строка



Значение по умолчанию : 100%



options/minheight



Минимальная высота блока предварительного просмотра в пикселях или процентах.



Тип : Число, Строка



Значение по умолчанию :null



options/minwidth



Минимальная ширина блока предварительного просмотра в пикселях или процентах.



Тип : Число, Строка



Значение по умолчанию :null



options/nav



Вариация миниатюр в навигации.



Возможные значения :



    
  
  • dots: точки в стиле iPhone
    • 
  
  • thumbs: миниатюры
    • 
  
  • false: ничего такого
    • 




По умолчанию :dots



options/navdir



Направление скольжения миниатюр.



Возможные значения :



    
  
  • vertical
    • 
  
  • horizontal
    • 




options/navarrows



Отображение навигационных стрелок для миниатюр.



Тип : логическое значение



options/navtype



Скользящий тип миниатюр.



Возможные значения :



    
  
  • slides: на кнопке со стрелкой нажмите на ленту с большими пальцами
    • 
  
  • thumb: на кнопке со стрелкой щелчок пальцами показывает, что лента сдвигается на один большой палец
    • 




options/ratio



Ширина делится на высоту. Рекомендуется, если вы устанавливаете ширину в процентах.



Тип : Число, Строка



Значение по умолчанию : рассчитывается из widthи height.



options/showCaption



Включить просмотр подписи в предварительном просмотре. Может быть для инициализации для конкретного изображения. Может работать глобально.



Тип : логическое значение



options/startindex



Номер индекса изображения, которое отображается после инициализации галереи.



Тип : Номер



Значение по умолчанию :0



options/swipe



Перемещение между изображениями предварительного просмотра, проводя влево и вправо.



Тип : логическое значение



Значение по умолчанию :true



options/thumbwidth



Ширина миниатюр.



Тип : Число, Строка



options/thumbheight



Высота миниатюр.



Тип : Число, Строка



options/transition

Настроить параметры галереи в view.xml



Параметры галереи и лупы могут быть установлены в view.xmlконфигурационном файле темы. Файл условно находится в <theme_dir>/etc.



Общие параметры галереи устанавливаются следующим образом:



<var name="gallery">
    <var name="%option1%">%option1_value%</var>
    <var name="%option2%">%option2_value%</var>
...
</var>

копия



Параметры полноэкранного режима и точки останова устанавливаются аналогичным образом:



<var name="fullscreen">
    <var name="%fullscreen_option1%">%option1_value%</var>
    <var name="%fullscreen_option2%">%option2_value%</var>
...
</var>
<var name="breakpoints">
    <var name="%breakpoints_option1%">%option1_value%</var>
    <var name="%breakpoints_option2%">%option2_value%</var>
...
</var>

копия



Для иллюстрации настройки галереи в view.xml, вы можете сослаться на view.xml пустой темы .



Галерея API



Методы галереи размещаются в хранилище данных galleryобъекта. Для инициализации API в объекте галереи вызовите dataметод с galleryаргументом as. Иллюстрация следует:



var api = $(element).data('gallery');

//or

var api = $('[data-gallery-role="gallery"]').data('gallery');

копия



Этот метод возвращает объект JS, который содержит    
 функции API .



Чтобы убедиться, что галерея полностью сформирована, оберните ваш код    
 функцией обработчика события и добавьте его в gallery:loadedсобытие:



$(element).on('gallery:loaded', function () {
    var api = $(element).data('gallery');
    /* api methods calls */
});

копия



Затем для вызова метода используйте следующую запись:



api.%method_name%();

копия



Все доступные методы перечислены в следующем абзаце.

first()



Отображает первое изображение предварительного просмотра.



last()



Отображает последнее изображение предварительного просмотра.



next()



Идентификатор изображения передается в качестве аргумента. Отображение следующего изображения предварительного просмотра.



Если передается последний идентификатор изображения, поведение зависит от того, включен ли цикл :



    
  
  • Если цикл включен, отображается первое изображение.
    • 
  
  • если петля отключена, отображаемое изображение не изменяется.
    • 




prev()



Отображение предыдущего изображения предварительного просмотра.



Если передается первый идентификатор изображения, поведение зависит от того, включен ли цикл :



    
  
  • если цикл включен, отображается последнее изображение.
    • 
  
  • если петля отключена, отображаемое изображение не изменяется.
    • 




seek()



Отображает изображение с указанным index.



Не обновляет предварительный просмотр, если аргумент недействителен.



Поведение:



    
  
  • seek(0) не отображается предварительный просмотр
    • 
  
  • seek(1) отображает первое изображение
    • 
  
  • seek(-1) отображает последнее изображение
    • 
  
  • Если аргумент является числом, большим, чем количество элементов, или меньше, чем «- (количество элементов)», галерея показывает модуль следующего деления: общее количество изображений, деленное на номер аргумента.
    • 




updateData()



Добавить новые предметы в галерею.



Пример:


api.updateData([{
    img: 'image1.jpg',
    thumb: 'thumb1.jpg',
    caption: 'caption'
}]);

копия



updateOptions()


Обновляет параметры активной точки останова или параметры галереи по умолчанию, если активной точки останова нет.



Пример:


api.updateOptions([{
    nav: 'dots'
}]);
